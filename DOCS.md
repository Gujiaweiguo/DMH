# DMH 数字营销枢纽 - 技术文档 (MVP 版本)

## 1. 项目概览
DMH (Digital Marketing Hub) 是一个轻量级、高性能的数字营销中台。

### 核心价值
- **极速部署**：1分钟内完成营销活动上线。
- **动态表单**：支持灵活的数据采集，无需后端改动。
- **实时激励**：支付即结算，显著提升推广员积极性。
- **无缝集成**：内置外网数据库适配器。

---

## 2. 技术架构

### 2.1 前端栈 (Admin & H5)
- **框架**：Vue 3.x (Composition API)
- **样式**：Tailwind CSS
- **图标**：Lucide Vue Next
- **移动端模拟**：Uni-app 风格 H5 容器

### 2.2 后端 (Business Logic)
- **逻辑模型**：基于 Go-Zero 的微服务架构。
- **核心逻辑**：
    - `api.createOrder`: 封装了事务处理、奖励计算、以及同步任务入队。
    - `SyncAdapter`: 模拟异步外网直连驱动。

---

## 3. 业务规则说明 (Business Rules)

### 3.1 实时奖励结算逻辑
1. **触发时机**：支付回调成功。
2. **逻辑**：推荐人余额在 2s 内自动更新。
3. **安全性**：同一活动/同一手机号限报一次。

---
