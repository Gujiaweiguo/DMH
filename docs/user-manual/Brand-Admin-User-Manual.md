## 系统登录

### 访问系统

1. 打开浏览器，访问品牌管理后台：`https://admin.dmh.com`
2. 输入分配给您的用户名和密码
3. 点击"登录"按钮

![品牌管理员登录](../images/brand-admin-login.png)

### 首次登录

首次登录时，系统会要求您：

1. **修改初始密码**: 设置符合安全策略的新密码
2. **完善个人信息**: 填写真实姓名、邮箱等信息
3. **查看权限范围**: 了解分配给您的品牌列表

### 登录后界面

登录成功后，您将看到：

- **顶部导航栏**: 显示用户信息和快捷操作
- **左侧菜单**: 根据您的权限显示可用功能
- **主工作区**: 显示当前选择的功能页面
- **品牌切换器**: 快速切换管理的品牌（如果管理多个品牌）

## 品牌信息管理

### 查看品牌信息

在左侧菜单中选择"品牌管理" > "品牌信息"，查看您管理的品牌详情：

![品牌信息页面](../images/brand-info.png)

#### 品牌信息包含

- **基本信息**
  - 品牌名称
  - 品牌Logo
  - 品牌描述
  - 创建时间
  - 最后更新时间

- **统计概览**
  - 总活动数量
  - 活跃活动数量
  - 总参与用户数
  - 本月新增用户

### 编辑品牌信息

点击"编辑品牌信息"按钮，可以修改：

![编辑品牌信息](../images/edit-brand-info.png)

#### 可编辑字段

1. **品牌名称**
   - 3-50个字符
   - 支持中英文和数字
   - 不能与其他品牌重名

2. **品牌Logo**
   - 支持JPG、PNG格式
   - 建议尺寸：200x200像素
   - 文件大小不超过2MB

3. **品牌描述**
   - 最多500个字符
   - 支持富文本格式
   - 可以包含品牌理念、特色等

#### 保存更改

1. 填写完信息后点击"保存"
2. 系统会验证信息格式
3. 保存成功后会显示确认消息
4. 更改会立即生效

### 品牌状态管理

品牌状态影响所有相关功能：

- **活跃状态**: 品牌正常运营，所有功能可用
- **暂停状态**: 品牌暂停运营，部分功能受限

**注意**: 品牌管理员无法修改品牌状态，如需变更请联系平台管理员。

## 品牌素材管理

### 素材库概览

在"品牌管理" > "素材库"中管理品牌的所有数字资产：

![素材库页面](../images/brand-assets.png)

#### 素材分类

- **Logo素材**: 品牌标识、图标
- **Banner素材**: 宣传横幅、海报
- **产品素材**: 产品图片、展示图
- **营销素材**: 广告图、推广素材
- **文档素材**: 品牌手册、说明文档

### 上传素材

点击"上传素材"按钮添加新素材：

![上传素材](../images/upload-asset.png)

#### 上传步骤

1. **选择文件**
   - 点击上传区域或拖拽文件
   - 支持批量上传
   - 实时显示上传进度

2. **填写信息**
   - 素材名称（必填）
   - 素材分类（必选）
   - 素材标签（可选）
   - 素材描述（可选）

3. **确认上传**
   - 检查信息无误后点击"确认上传"
   - 系统自动处理和存储文件
   - 生成缩略图和预览

#### 支持的文件格式

| 类型 | 格式 | 大小限制 | 说明 |
|------|------|----------|------|
| 图片 | JPG, PNG, GIF, WebP | 10MB | 自动生成缩略图 |
| 视频 | MP4, AVI, MOV | 100MB | 自动生成预览图 |
| 文档 | PDF, DOC, PPT, XLS | 20MB | 支持在线预览 |

### 管理素材

#### 查看素材

- **列表视图**: 显示素材详细信息
- **网格视图**: 以缩略图形式展示
- **搜索过滤**: 按名称、分类、标签搜索

#### 编辑素材

点击素材的"编辑"按钮可以：

1. **修改基本信息**
   - 更改素材名称
   - 调整分类和标签
   - 更新描述信息

2. **替换文件**
   - 上传新版本文件
   - 保留原有信息和链接
   - 自动更新预览

3. **设置权限**
   - 公开：所有人可见
   - 私有：仅品牌团队可见

#### 删除素材

删除素材前请注意：

1. **检查使用情况**: 确认素材未在活动中使用
2. **备份重要文件**: 删除后无法恢复
3. **确认删除**: 需要二次确认操作

### 素材组织

#### 文件夹管理

创建文件夹来组织素材：

1. 点击"新建文件夹"
2. 输入文件夹名称
3. 拖拽素材到文件夹中
4. 支持多级文件夹结构

#### 标签系统

使用标签快速分类和查找：

- **颜色标签**: 按主色调分类
- **用途标签**: 按使用场景分类
- **时间标签**: 按创建时间分类
- **自定义标签**: 根据需要自定义

## 常见问题

### Q1: 如何切换管理的品牌？

**A**: 如果您管理多个品牌：
1. 点击页面顶部的品牌切换器
2. 从下拉列表中选择要管理的品牌
3. 页面会自动刷新显示对应品牌的数据

### Q2: 上传的素材无法显示怎么办？

**A**: 请检查以下几点：
1. 文件格式是否支持
2. 文件大小是否超出限制
3. 网络连接是否稳定
4. 浏览器是否支持该格式

### Q3: 活动数据统计不准确怎么办？

**A**: 数据统计可能有延迟：
1. 等待5-10分钟后刷新页面
2. 检查活动时间设置是否正确
3. 确认统计时间范围设置
4. 如问题持续，请联系技术支持

### Q4: 如何提高活动参与率？

**A**: 建议采取以下措施：
1. 优化活动页面设计
2. 设置有吸引力的奖品
3. 简化参与流程
4. 增加社交分享功能
5. 选择合适的推广渠道

### Q5: 忘记密码怎么办？

**A**: 可以通过以下方式重置：
1. 在登录页面点击"忘记密码"
2. 输入注册邮箱或手机号
3. 按照提示完成身份验证
4. 设置新密码

### Q6: 如何申请更多品牌管理权限？

**A**: 需要联系平台管理员：
1. 准备权限申请说明
2. 说明业务需求和理由
3. 通过邮件或系统消息联系管理员
4. 等待审核和分配

### Q7: 素材库空间不够用怎么办？

**A**: 可以采取以下措施：
1. 清理不再使用的素材
2. 压缩大文件减少空间占用
3. 联系管理员申请扩容
4. 使用外部CDN存储大文件

### Q8: 活动审核被拒绝怎么办？

**A**: 请按以下步骤处理：
1. 查看审核反馈意见
2. 根据要求修改活动内容
3. 重新提交审核
4. 如有疑问联系审核人员
